GDD211 Lab 1 - Advanced Animation - Quiz

Please recreate this scene in Unity.
You can find the project template here: https://github.com/ProfessorBlake/GDD211_Lab1_Quiz


Puppy 1 (Top) - Animations with transitions
  • Idle animation plays when not moving.
  • Walk animation plays when moving.
  • Animator transitions between Idle and Walk by checking condition of a parameter (your choice of which type).

Puppy 2 (Middle) - Animations with Blend Tree
  • Idle animation plays when not moving.
  • Walk animation plays when moving.
  • Animator uses blend tree to control which animation is playing.

Puppy 3 (Bottom) - Animations with Animation Event
  • IdleEvents animation plays when not moving.
  • Walk animation plays when moving.
  • Add an animation event to the IdleEvents animation clip which calls the PuppyMagicSpell function in the AnimationEvent script.
